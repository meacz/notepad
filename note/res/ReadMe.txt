	  ===========================
	     notepad v1.1帮助文档
	  ===========================



快捷键：

	Ctrl + C    复制（支持无选中对当前行复制）
	Ctrl + X    剪切（支持无选中对当前行剪切）
	Ctrl + V    粘贴	
	Ctrl + Z    撤销（支持无限制撤销）
	Ctrl + Y    重做（支持无限制重做）
	Ctrl + J    在当前行插入新的空白行
	Ctrl + D    复制当前行并插入到下一行
	Ctrl + H    替换
	Ctrl + F    查找
	Ctrl + G    跳转 
	Ctrl + A    全选
	F5	    时间/日期
	Ctrl + ↓   将当前行向下移动一行
	Ctrl + ↑   将当前行向上移动一行
	Ctrl + N    新建文件
	Ctrl + O    打开文件
	Ctrl + S    保存文件



其它功能：
	
	格式 -> 字体：
		设置文档使用的字体样式
	
	查看 -> 格式设置：
		预览或设置当前文档使用的样式


	帮助 -> 设置：
		自动换行，配置文件，其它样式颜色设置


支持多选（按住Ctrl键，使用鼠标对文本进行选择）
