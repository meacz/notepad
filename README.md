### 简介：
使用sintilla和duilib编写的小工具，功能和window记事本类似。  
支持的快捷键：  
无选中复制(Ctlr+C)、剪切(Ctrl+X)  
Ctrl + D  将当前行复制到下一行  
Ctrl + L  删除当前行  
Ctrl + J  在当前行插入新的空白行  

![](img/note.png)

### 编译：  
需要安装Visual Studio 2017，默认项目中已经配置好所需要的依赖库，编译即可。  
配置：  
Duilib_Static（Debug、Release、UnicodeDebug、UnicodeRelease）  
Scintilla（Debug、Release）  
w（Debug、Release）  

### 运行环境：  
window7、10
